CSS notes

Inline vs block display option

A block element will start on a new line and will fill the entire length of a page. Examples of elements with this default type are: <article>, <div>,<p>. Inline elements do not start on a new line and contain only data and other inline elements. Block elements may contain inline elements. Examples of inline elements include: <a>, <code>,<img>. There are two other CSS display options: inline-block and none. Inline-block has properties of block element, but does not begin on a new line. None will make the object disappear!

Box model

WTF is the box model?

The box bodel is a conceptual model that all elements of a page adhere to. It renders all elements in a rectangular box, allowing for width, height, padding, borders and margins to be manipulated.
